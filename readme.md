## Instalación

Debéis ejecutar
```bash
npm i
```

## Compilar
Podéis compilar todos los archivos y observar los cambios con:
```bash
npm run watch:all
```

o si queréis compilar un ejercicio en concreto (ejemplo, ejercicio 3)
```bash
npm run compile3
```

## Ejecutar
Para ejecutar del 1 al 7 podéis usar (cambiando de número)
```bash
node 2.js
```
